# Learning Organizer API

 An API that let the user add their own learning resources, articles, etc.
It's built to minimize distractions and chaos, when you have so many things to
study one time you get distracted and lost so there's a limit of adding resources 
there's three status: 

## Current:
Which it is what you studying or learning at the moment the limit of this status
is 3 resources it's great for focus and makes you priorities the resources.

## Library:
This the state where you save some resources for later it has a limit of
10 resources.

## Finished:
The state that you can view your own done list of resources.

Each state has a name, link, status that you can update them.
